Long Haul Walk

About

Long Haul Walk is an indie game that has a main gameplay motive of adventure. The theme is nature and this game wants to promote a relaxing mood for player. The game is believed to be really passive to the player, that means player will not be attacked from any entities in the game. Except other player, physics and some objects. From other player it also means game is expected to have multiplayer. And this expectation is somewhat fulfilled. Otherwise game is categorized as adventures and role-playing.
